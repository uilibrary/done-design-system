Serve & Build

Documentation of development server and build process

Installation

Done uses Gulp for development serve and build process

  • 1. Make sure NodeJs installed on your computer.
  • 2. Open terminal and navigate to root /done-design-system directory and run npm install to install dependencies.

Development server

Run npm start to start local development server. Open http://localhost:3000/ in browser.

Build

Run npm run build to build your project. All files including compiled & minified css, js files will be stored in /dist directory.

JavaScript dependencies sometimes get out of sync and inexplicable bugs start to happen. Follow these steps to give a fresh start to your development environment:

  • 1. The installed npm version must be at least v6.9.0 You can update your npm with: npm install npm -g (sudo may be required).
  • 2. Re-install dependencies: rm -Rf node_modules && npm install
  • 3. Run npm start
  • If this did not work, try running npm cache clean and repeat the above steps.
